Paia Support

Support information for the Paia iOS app.

Contact

For help with Paia, contact: cjurkiewicz@gmail.com

Common Questions

  • Paia requires a one-time on-device model download before chat is available.
  • After the model download completes, core chat and personal document retrieval run locally on device.
  • If voice input is unavailable, the current device or language may not support on-device speech recognition. Text chat will still work.
  • Imported documents and chat history are stored locally in the app container on the device.

Troubleshooting

  • If the model download fails, check network connectivity and try again.
  • If responses are slow or unstable, use a device with sufficient available memory.
  • If voice mode is unavailable, continue with text input instead.
